## 📁 文件格式

### JSONL格式示例
```jsonl
{"timestamp": "2026-02-02 02:00:00", "date": "2026-02-02", "analysis_time": "02:00:00", "color_counts": {"green": 0, "red": 12, "yellow": 0, "blank": 0, "blank_ratio": 0.0}, "signal": "做空", "description": "🔴 只有红色柱子，预判下跌行情，建议做空。操作提示：相对高点做空", "is_temp": false, "is_final": true}
```

### 字段说明
- `timestamp`: 完整时间戳（北京时间）
- `date`: 日期
- `analysis_time`: 分析时间（通常为02:00:00）
- `color_counts`: 柱状图颜色统计（绿/红/黄/空白）
- `signal`: 市场信号（做空/低吸/等待新低等）
- `description`: 详细描述和操作建议
- `is_temp`: 是否为临时数据
- `is_final`: 是否为最终预判
- `regenerated`: （可选）是否为重新生成的数据
