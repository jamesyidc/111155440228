# 0-2点行情预判逻辑优化说明

## 📋 修改日期
**2026-02-25**

## 🎯 修改目标
优化0-2点柱状图分析的行情预判逻辑，增加**红色+黄色柱子数量**作为判断阈值，提高预判准确性。

---

## 📊 核心修改

### 🔴 情况2：等待新低（高点做空）

#### ✅ 新逻辑（修改后）
满足以下任一条件：

1. **有绿色柱子 + 有红色柱子 + 有黄色柱子**，且 **（红+黄）>= 3根（>=25%）**
   - 信号：等待新低
   - 操作提示：高点做空
   - TG通知：发送"等待新低"提醒

2. **只有绿色柱子 + 黄色柱子（无红色）**，且 **黄色 >= 3根**
   - 信号：等待新低
   - 操作提示：高点做空
   - TG通知：发送"等待新低"提醒

|------|----|----|----| ------|----------|
| 2026-02-06 | 4 | 7 | 1 | 8根 | ✅ 等待新低（高点做空）|
| 2026-02-09 | 1 | 8 | 1 | 9根 | ✅ 等待新低（高点做空）|
| 2026-02-18 | 7 | 3 | 1 | 4根 | ✅ 等待新低（高点做空）|
| 2026-02-21 | 3 | 7 | 2 | 9根 | ✅ 等待新低（高点做空）|

---

### 🟢 情况1：低吸（低点做多）

#### ✅ 新逻辑（修改后）
满足以下任一条件：

1. **有绿色柱子 + 有红色柱子 + 没有黄色柱子**
   - 信号：低吸
   - 操作提示：低点做多
   - TG通知：发送"低吸"提醒
   - 说明：红色区间为低吸机会

2. **有绿色柱子 + 有红色柱子 + 有黄色柱子**，但 **（红+黄）< 3根（<25%）**
   - 信号：低吸
   - 操作提示：低点做多
   - TG通知：发送"低吸"提醒
   - 说明：红色+黄色柱子数量不足，信号不明显，按低吸处理

#### 📌 示例数据

|------|----|----|----| ------|----------|
| 2026-02-10 | 8 | 4 | 0 | 4根 | ✅ 低吸（低点做多）|
| 2026-02-12 | 9 | 3 | 0 | 3根 | ✅ 低吸（低点做多）|
| 2026-02-25 | 10| 1 | 1 | 2根 | ✅ 低吸（低点做多）|

## 🔄 逻辑对比

### 🔴 旧逻辑（修改前）
```
情况2: 有绿 + 有红 + 有黄  →  等待新低
情况1: 有绿 + 有红 + 无黄  →  低吸
```

**问题**：
- 只要有黄色柱子，就判断为"等待新低"
- 没有考虑红色+黄色柱子的数量
- 可能导致误判（例如：绿10 红1 黄1 仍判断为"等待新低"）

### 🟢 新逻辑（修改后）
```
情况2: 
  - 有绿 + 有红 + 有黄，且（红+黄）>=3根  →  等待新低
  - 有绿 + 黄色（无红），且黄色>=3根       →  等待新低

情况1: 
  - 有绿 + 有红 + 无黄                   →  低吸
  - 有绿 + 有红 + 有黄，但（红+黄）<3根  →  低吸
```

**优势**：
- ✅ 增加数量阈值（3根 = 25%），提高判断准确性
- ✅ 避免因少量黄色柱子导致的误判
- ✅ 更符合实际市场行情特征

## 📁 修改文件

### 1. 核心逻辑文件
**文件**: `scripts/regenerate_all_predictions.py`

**修改行数**: 122-145

**关键代码**:
```python
# 情况2: 有绿+有红+有黄，且（红+黄）>= 3根（>=25%）→ 等待新低
elif green > 0 and red > 0 and yellow > 0 and (red + yellow) >= 3:
    signal = "等待新低"
    description = f"🟢🔴🟡 有绿有红有黄，红色+黄色共{red + yellow}根(>=3根)，可能还有新低，建议等待。操作提示：高点做空"

# 情况1: 有绿+有红+无黄 OR 有绿+有红+有黄但（红+黄）< 3根（<25%）→ 低吸
elif green > 0 and red > 0 and (yellow == 0 or (red + yellow) < 3):
    if yellow == 0:
        signal = "低吸"
        description = "🟢🔴 有绿有红无黄，红色区间为低吸机会。操作提示：低点做多"
    else:
        signal = "低吸"
        description = f"🟢🔴🟡 有绿有红有黄，但红色+黄色共{red + yellow}根(<3根)，红色区间为低吸机会。操作提示：低点做多"

# 情况8: 只有绿色+黄色（无红色），且黄色>=3根 → 等待新低
elif green > 0 and yellow >= 3 and red == 0:
    signal = "等待新低"
    description = f"🟢🟡 只有绿色和黄色（黄色{yellow}根>=3根），可能还有新低，建议等待。操作提示：高点做空"
```
