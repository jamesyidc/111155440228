## 💻 技术细节

### HTML5 Details/Summary
```html
<details>              <!-- 折叠容器 -->
  <summary>标题</summary>  <!-- 可点击标题 -->
  <div>内容</div>        <!-- 折叠内容 -->
</details>
```

**优点**:
- ✅ 原生HTML5标签，无需JavaScript
- ✅ 浏览器原生支持
- ✅ 语义化，SEO友好
- ✅ 键盘可访问（Enter/Space切换）
- ✅ 屏幕阅读器支持

### CSS动画
```css
/* 箭头旋转 */
summary::before {
  content: '▶';
  transition: transform 0.3s;
}

details[open] summary::before {
  transform: rotate(90deg);
}

/* 边框和背景变化 */
details {
  background: #f8fafc;
  transition: all 0.3s;
}

details[open] {
  background: #fff;
  border-color: #667eea;
}
```
