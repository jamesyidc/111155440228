### 预期行为
根据规则：

- **红+空白且空白<25%** → 才是**诱空试盘抄底**

## 🔍 问题排查

### 检查代码逻辑

**原始代码**（第108行）：
```python
# 情况5: 诱空试盘抄底
elif green == 0 and yellow == 0 and red > 0 and blank_ratio < 25:
    signal = "诱空试盘抄底"
```

**问题分析**：
1. 当 `blank = 0`（纯红色）时
2. `blank_ratio = 0 / 12 * 100 = 0%`
3. `0% < 25%` 条件满足 ✅
4. 被错误匹配为"诱空试盘抄底" ❌

### 根本原因

**逻辑缺陷**：
- 条件只检查了 `blank_ratio < 25`
- 没有检查是否真的**有空白**（`blank > 0`）
- 导致纯红色（`blank = 0`）也被匹配

---

## ✅ 解决方案

### 修复代码

**修改后**（第108行）：
```python
# 情况5: 诱空试盘抄底（必须有空白且空白<25%）
elif green == 0 and yellow == 0 and red > 0 and blank > 0 and blank_ratio < 25:
    signal = "诱空试盘抄底"
    description = f"⚪🔴 只有红色和空白（空白占比{blank_ratio:.1f}%<25%），诱空行情，可以试盘抄底。操作提示：低点做多"
```

**关键改动**：
- ✅ 增加 `blank > 0` 条件
- ✅ 确保只有**真正有空白**且**空白占比<25%**才会被判定为"诱空试盘抄底"
- ✅ 纯红色（`blank = 0`）会被后续的情况3捕获，判定为"做空"

### 完整逻辑

#### 情况5（诱空试盘抄底）
**条件**：
- 红色>0，绿色=0，黄色=0
- **有空白（blank>0）且 空白占比<25%**

**信号**：诱空试盘抄底  
**操作**：低点做多  

**示例**：
- 红11 空白1（8.3%）✅
- 红10 空白2（16.7%）✅

#### 情况3（做空）
**条件**：
- 红色>0，绿色=0，黄色=0
- **纯红色（blank=0）或 空白占比≥25%**

**信号**：做空  
**操作**：相对高点做空  

**示例**：
- 红12 空白0（纯红色）✅
- 红8 空白4（33.3%）✅
- 红6 空白6（50%）✅
