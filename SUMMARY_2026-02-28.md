## 问题1：27币涨跌幅之和线条不显示

### 问题描述
用户多次反馈：币种追踪器页面中，"27个币涨跌幅之和"的线条**不显示**或**刷新时短暂出现然后马上消失**。

### 根本原因
经过多次调试，最终发现真正的根本原因：

**series[0]（27币涨跌幅之和）缺少 `lineStyle` 配置**

在 `templates/coin_change_tracker.html` 第6774-6795行：
```javascript
series: [{
    name: '27币涨跌幅之和',
    type: 'line',
    yAxisIndex: 0,
    smooth: true,
    data: cleanedChanges,
    areaStyle: { ... },  // ✅ 有填充样式
    // ❌ 缺少 lineStyle！导致线条不可见
}]
```

### 修复方案
为series[0]添加完整的lineStyle配置：

```javascript
series: [{
    name: '27币涨跌幅之和',
    type: 'line',
    yAxisIndex: 0,
    smooth: true,
    data: cleanedChanges,
    lineStyle: {  // ✅ 新增
        color: '#3B82F6',  // 蓝色
        width: 2
    },
    areaStyle: { ... },
}]
```

### 修复效果
1. ✅ 27币涨跌幅之和的**蓝色线条**清晰可见
2. ✅ 线条下方有**蓝色渐变填充区域**
3. ✅ 线条在页面刷新后**持续显示**
4. ✅ 切换日期、窗口resize等操作后线条**保持显示**
5. ✅ 与RSI线条（灰色虚线）同时显示，互不干扰

## 问题2：8种情况自动化预判逻辑优化

### 用户需求
1. **情况3【做空】**：允许做空不允许做多
   - 若空白柱子占≥25%仍做空
   - 操作提示：相对高点做空

2. **情况5【诱空】**：允许做多不允许做空
   - 0点-2点红色+空白且空白占比<25%则判断为诱空
   - TG发送消息：试盘抄底
   - 操作提示：低点做多

### 修复方案

#### 调整判断优先级
原来的顺序会导致情况5和情况3冲突，调整为：

1. 情况6：全空白（空头强控盘）
2. 情况4：全绿色（诱多不参与）
3. **情况5：红色+空白且空白<25% → 诱空试盘抄底**
4. **情况3：红色+空白且空白>=25% OR 只有红色 → 做空**

#### 修改后的逻辑

**情况5（诱空）**：
```python
# 只有红色+空白（无绿无黄）且空白占比<25%
elif green == 0 and yellow == 0 and red > 0 and blank_ratio < 25:
    signal = "诱空试盘抄底"
    description = f"⚪🔴 只有红色和空白（空白占比{blank_ratio:.1f}%<25%），诱空行情，可以试盘抄底。操作提示：低点做多"
```

**情况3（做空）**：
```python
# 只有红色或红色+空白且空白>=25%
elif red > 0 and green == 0 and yellow == 0:
    if blank == 0:
        signal = "做空"
        description = "🔴 只有红色柱子，预判下跌行情，建议做空。操作提示：相对高点做空"
    else:  # blank_ratio >= 25%（因为<25%已被情况5捕获）
        signal = "做空"
        description = f"🔴⚪ 红色+空白（空白占比{blank_ratio:.1f}%>=25%），预判下跌行情，建议做空。操作提示：相对高点做空"
```

### 测试结果

| 日期 | 颜色分布 | 空白占比 | 预判结果 | 状态 |
|------|---------|---------|---------|------|
| 2026-02-02 | 红12 绿0 黄0 | 0% | 诱空试盘抄底 | ✅ |
| 2026-02-13 | 红6 绿0 黄0 空白6 | 50% | 做空 | ✅ |
| 2026-02-27 | 红12 绿0 黄0 | 0% | 诱空试盘抄底 | ✅ |
| 2026-02-28 | 红8 绿0 黄0 空白4 | 33.3% | 做空 | ✅ |

### 修复效果
1. ✅ 明确区分"做空"和"诱空"两种情况
2. ✅ 空白占比<25%：诱空试盘抄底（低点做多）
3. ✅ 空白占比>=25%：做空（相对高点做空）
4. ✅ 避免了情况3和情况5的冲突
5. ✅ 所有25个日期的预判文件已重新生成
